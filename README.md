# Gampa Rithika — Portfolio

A modern, professional portfolio website built with React.

## 🚀 Quick Start

### 1. Install dependencies
```bash
npm install
```

### 2. Run locally
```bash
npm run dev
```
Open http://localhost:5173 in your browser.

### 3. Build for production
```bash
npm run build
```

## 🌐 Deploy

### Vercel (Recommended)
```bash
npm install -g vercel
npm run build
vercel --prod
```

### Netlify
```bash
npm run build
# Drag & drop the dist/ folder to netlify.com/drop
```

## ✅ Contact Form Setup (Formspree)
1. Go to https://formspree.io and sign up free
2. Create a new form → copy your Form ID
3. In App.jsx, replace `xvgaqwno` with your actual Form ID

## 🔗 Links to Update in App.jsx
- GitHub: Replace `https://github.com/gamparithika` with your full URL
- LinkedIn: Already set to your profile
